# Catatan Keuangan Pribadi

Aplikasi Android sederhana untuk mencatat pemasukan dan pengeluaran harian. Dibuat menggunakan Kotlin dengan tujuan membantu pengguna dalam mengelola keuangan secara praktis.

## 🚀 Fitur Utama
- Tambah pemasukan
- Tambah pengeluaran
- Menampilkan daftar transaksi
- Perhitungan total saldo
- UI sederhana dan user-friendly

## 🛠️ Teknologi yang Digunakan
- Kotlin
- Android Studio
- (Opsional: SQLite / Room Database / Firebase)

## 📂 Struktur Project
- MainActivity (Halaman utama)
- Activity/Form input data
- Adapter (RecyclerView/ListView)
- Database/Model

## ⚙️ Cara Menjalankan
1. Clone repository ini
2. Buka di Android Studio
3. Sync Gradle
4. Jalankan di emulator / device
